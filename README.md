# 📖 Overview

AI Photo Creator is a lightweight and interactive web application that transforms text descriptions into AI-generated artwork.

The application focuses on providing a clean user experience while allowing users to customize image styles, resolutions, and prompt quality before generating images.

---

# ✨ Features

- 🎨 Generate AI images from text prompts
- 🖌️ Multiple artistic styles
- 📐 Adjustable image width and height
- ✨ Magic Enhance for detailed prompts
- 🎲 Surprise Me with creative prompt ideas
- 📥 Download generated images as PNG
- ⚡ Fast and responsive Streamlit interface
- 🎯 Simple and beginner-friendly design

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sanjayduduka45/ai-photo-creator.git
```

Navigate into the project

```bash
cd ai-photo-creator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

# 💡 Core Functionalities

## 🎨 AI Image Generation

Generate AI-powered artwork by describing your imagination using natural language.

---

## 🖌 Art Style Selection

Choose from multiple artistic styles to personalize your generated image.

Examples include:

- Photorealistic
- Anime
- Sketch
- Vintage Victorian
- 3D Render

---

## 📐 Image Resolution

Adjust image dimensions before generation using intuitive sliders.

---

## ✨ Magic Enhance

Automatically enriches prompts with descriptive keywords to improve image quality.

---

## 🎲 Surprise Me

Generate creative artwork instantly from a curated collection of random prompts.

---

## 📥 Image Download

Save generated artwork directly as a PNG image.
