# 🎨 AI Photo Creator

### Turn your imagination into AI-generated artwork with a single prompt.

A modern **Streamlit** application that transforms text prompts into stunning AI-generated images using the **Pollinations AI** image generation API. Customize artistic styles, image resolution, and creative enhancements while enjoying a clean and intuitive interface.

# ✨ Features

## 🎨 AI Image Generation

Generate high-quality AI artwork from natural language prompts.

---

## 🖌 Multiple Art Styles

Choose different artistic styles such as

* Photorealistic
* Anime
* Sketch
* Vintage Victorian
* 3D Render

---

## 📐 Custom Image Resolution

Generate images in different dimensions using adjustable width and height sliders.

---

## ✨ Magic Enhance

Automatically enriches prompts with professional keywords for more detailed and visually appealing outputs.

---

## 🎲 Surprise Me

Generate completely random creative artwork with a single click using curated prompts.

---

## 📥 One-Click Download

Download generated images directly as PNG files with dynamic filenames.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sanjayduduka45/ai-photo-creator.git
```

Navigate to the project folder

```bash
cd ai-photo-creator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
